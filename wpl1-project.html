Werkplekleren 1 Project: Dashboard

Projectomschrijving

Dit is mijn project voor het vak Werkplekleren 1. In dit project heb ik een dashboard ontwikkeld dat mijn vaardigheden, softwarekennis en persoonlijk rooster op een visueel aantrekkelijke manier weergeeft. Het dashboard is ontworpen met tools zoals Photoshop, Illustrator, Figma, HTML, CSS en JavaScript, en richt zich op het verbeteren van de gebruikerservaring (UI/UX).In dit project volgden we: POP-sessies (3x) Werkveldverkenning - Seminaries Werkveldverkenning - Vacatures Dashboard CV + Concept-Poster Visual Design Portfolio

Projectdoelen

  • Verbeteren van visuele presentatievaardigheden.
  • Toepassen van UI/UX-ontwerpprincipes.
  • Versterken van mijn kennis in webontwikkeling (HTML, CSS, JavaScript).

Gebruikte Tools

  • Photoshop
  • Illustrator
  • Figma
  • HTML
  • CSS
  • JavaScript

Resultaat

Het eindresultaat is een gebruiksvriendelijk en responsief dashboard...
